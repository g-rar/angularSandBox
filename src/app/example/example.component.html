Ejemplo de ngFor

Al usar *ngFor como atributo de un elemento, se va a repetir ese elemento como en un forLoop. Todo el contenido dentro de dicha etiqueta se va a repetir del mismo modo. Una utilidad es por ejemplo, si se tiene un array de elementos y se quieren mostrar las propiedades de cada elemento en una tabla.

Por ejemplo: ‹tr *ngFor="let elem of listaEjemplo"› ...inserte aqui mas html... ‹/tr›
